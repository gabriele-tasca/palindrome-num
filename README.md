## palindrome-num
Determine if a number is palindrome or not. Works on strings or ints

### To run:
```
dotnet run --project palindrome/palindrome.csproj
```
Provide a number through standard input, which will be analyzed as a string.

Output:
```
Type your number
12344321
Result: True
```
